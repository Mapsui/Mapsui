This currently only works on MacOS. In Visual Studio 2019 for Mac or Rider. 

Following things need to be installed.

1. https://dotnet.microsoft.com/en-us/download
   download dotnet 6.0.200 SDK for x64 (for Visual Studio 2019 for Mac even on M1 Processors).
2. ios Workload
   sudo dotnet workload install ios
